# Vaultwarden

Single-container Bitwarden-compatible password manager. Stores everything in
`./data/db.sqlite3`; included in the daily restic offsite plus the B2 critical
set (vault loss is unrecoverable).

- Public domain: **bitwarden.example.com**
- Upstream: <https://github.com/dani-garcia/vaultwarden/wiki>
- Signups disabled — invite via the admin token interface.
